the scent library

For a final project, we were tasked with creating a website which somehow presented a collection of objects using JSON objects. This project was coded using HTML, CSS, and JQuery via Sublime Text.

Mimicking an online retail experience, I decided to create a website containing a collection of scents familiar or interesting to me. I wanted to translate my experiences of and memories associated with these scents into an interactive webpage everyone can play around with. Everytime you refresh the page, three random scents appear.

Making this project was very much an iterative process, starting with sketching a very rough wireframe that I loosely followed. I looked at various perfume retail websites to gain inspiration for how they represented smell visually. I chose to have each image pop up randomly on the page after each click to mimic the idea of scent particles being sprayed out. Click here to go to the website!
