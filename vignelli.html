vignelli zine

Using an excerpt from The Vignelli Canon, we were assigned to create a re-imagined zine or booklet based purely on the text. I had a lot of fun with this project as we were all given a lot of creative freedom, and I felt really inspired to make my voice as a designer present in this work. The text was an inspiration to me, and one of my favorite parts of this project was setting strict rules for myself. I only wanted the book to be in black, white, and red. All the images I chose to use somehow referred to the text on that page, usually through a visual pun. Applications Used: Adobe Indesign, Adobe Photoshop
